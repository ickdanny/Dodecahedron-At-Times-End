---------------------------------------------
Tutorial

Menu Navigation
z = select
x = back
left = back
right = select
up = go up
down = go down

Gameplay
shift = focus
z = attack
x = bomb
arrow keys = movement
esc = pause

Pause Menu
esc = unpause
z = select
left = go left
right = go right

Dialogue Screen
z = continue
x = skip

If game window does not fit your screen imput scaling ratio in frame_size_multi.
